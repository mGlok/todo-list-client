# todo-list-client
Vite, React, Typescript - ToDo List Client for my [ToDo List API](https://github.com/mGlok/todo-list-api)

## How to install:

```bash
  git clone https://github.com/mGlok/todo-list-client
  cd todo-list-client
  yarn
```

### After installation:

```bash
  yarn build
  yarn preview
```
